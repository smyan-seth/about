# Personal Portfolio Website

This is a simple, responsive personal portfolio website built using HTML, CSS, and JavaScript. It showcases sections like About, Education, Projects.

## 🚀 Features

- Fully responsive layout
- Mobile navigation hamburger menu
- Clean and modern design
- SEO and structured data support

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome (for icons)
